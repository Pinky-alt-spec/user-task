Task Project

This is an exciting project that runs on Python using Django Framework. The framework enables rapid development of secure, robust and maintainability for our Task project. The development consist of components that makes up both the front and backend.

  • 1. Model - I have used Model for data description in the database
  • 2. Views - Handles the display of users
  • 3. Templates - Controls the user Interaction
  • 4. Controller(Views) - Presentation layer for users
  • I have also included signals in the project. This causes user profile to be triggered when the user is registering. This prevents the user to create an account and then a profile. With Signals when the user registers, profile account is automatically created.
  • I have also used Django Signals, this simplify the implementation of our decoupled systems in various ways. They help us implement reusable applications and instead of reimplementing functionality severally, or modifying other parts of the system, we can just respond to signals without affecting other code
  • for eg. The user does not need to register and create a profile. With signals, when the user registers, the profile is automatically created.
